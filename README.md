# 🌦️ Smart Weather Advisor Lite

A minimal CLI-based weather assistant that generates outfit suggestions and a daily vibe score based on user input.

---

## ⚡ Tech Stack

`Python` `CLI` `No API` `Simulation`

---

## 🚀 Features

* Lightweight and fast
* "Feels like" temperature logic
* Outfit recommendations
* Simple mood scoring system

---

## ▶️ Run

```bash
python smart_weather.py
```
